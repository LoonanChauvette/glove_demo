### Running the Glove Demo

This project uses [**uv**](https://docs.astral.sh/uv/) for fast dependency management and reproducible environments.
You only need **Git** and **uv** installedâ€”everything else is handled automatically.

#### 1. Install prerequisites

Install Git (if not already installed)
https://git-scm.com/install/windows

Install uv
https://docs.astral.sh/uv/getting-started/installation/


#### 2. Clone the repository

```bash
git https://github.com/LoonanChauvette/glove_demo.git
cd glove_demo
```

#### 3. Run the project

```bash
# Run directly using uv
uv sync
uv run glove_demo
```
